# Huen's  Method
***
## Description
* Uses the huen's method with iteration and plots a t vs y plot.
## Inputs
*   dydt - differential equation
*   tspan - span on t
*   y0 - initial condition
*   h - step size
* -optional
*   es - stopping criterion percent
*   maxit - maximum number of iterations
## Outputs
*   plot of t and y
